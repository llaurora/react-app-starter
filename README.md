## Quick Start
~~~
$ git clone https://github.com/DlLucky/react-redux-app-init.git
$ cd react-redux-app-init
// 安装依赖
$ yarn install 或者 yarn  
// 启动
$ yarn run dev 
// 打包
$ yarn run build
~~~
